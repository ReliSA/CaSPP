# Installation Documentation Overview

The installation documentation is separated into two files according to the target audience.

---

## 1. User Installation

File:

```text
USER_INSTALLATION.md
```

This file is intended for regular users who only want to run the application.

It describes:

* basic requirements,
* running an existing executable,
* creating the executable using the provided build scripts,
* running the generated executable on Windows,
* running the generated executable on Linux / macOS,
* notes about Git, PowerShell script execution, and executable permissions.

---

## 2. Development Setup

File:

```text
DEVELOPMENT_SETUP.md
```

This file is intended for developers who want to work with the source code.

It describes:

* checking the Python version,
* cloning the repository,
* creating and activating a virtual environment,
* installing dependencies,
* building the application from source,
* running the built executable.

---

## Recommended Usage

Regular users should start with:

```text
USER_INSTALLATION.md
```

Developers should start with:

```text
DEVELOPMENT_SETUP.md
```
